# 🌤️ Vane — AI Телеграм-бот для прогнозу погоди

> Твій розумний погодний асистент на базі Gemini AI та WeatherStack.

---

## 📌 Про проект

**Vane** — це Telegram-чат-бот, який надає персоналізований прогноз погоди для будь-якого міста світу. Просто надішли боту назву міста — він отримає актуальні погодні дані та використає Google Gemini AI, щоб сформувати зручну та корисну відповідь: не лише погоду, а й практичні поради — що вдягнути, чи брати парасолю тощо.

### Як це працює:

1. Ти надсилаєш назву міста боту в Telegram.
2. **Gemini 2.5 Flash** знаходить координати цього міста.
3. Координати передаються до **WeatherStack API** для отримання поточних погодних даних.
4. Дані про погоду повертаються назад до **Gemini**, який форматує їх у читабельне повідомлення.
5. Ти отримуєш зрозумілий та дружній прогноз із порадами щодо одягу та активностей.

---

## 🎯 Цільова аудиторія

Vane буде корисним практично кожному:

- **Людям у дорозі** — дізнатись, що вдягнути, перш ніж виходити з дому.
- **Мандрівникам** — швидко перевірити погоду в незнайомому місті.
- **Студентам і працівникам** — зручна перевірка погоди без зайвих застосунків.
- **Активним користувачам Telegram** — усе в одному месенджері, без зайвих кроків.

Коротко кажучи — якщо ти користуєшся Telegram і хочеш знати погоду, Vane для тебе.

## 🚀 Початок роботи

### Передумови

Переконайся, що на твоєму комп'ютері встановлено **Node.js**.

### 1. Клонування репозиторію

```bash
git clone https://github.com/your-username/vane_tg_bot.git
cd vane_tg_bot
```

### 2. Встановлення залежностей

```bash
npm install @google/genai axios dotenv telegraf
```

### 3. Налаштування змінних середовища

Скопіюй файл-приклад і заповни свої дані:

```bash
cp .env.example .env
```

Відкрий `.env` і вкажи наступні ключі:

```env
TOKEN=твій_токен_телеграм_бота
GEMINI_API_KEY=твій_ключ_gemini
Url=http://api.weatherstack.com/current?access_key=твій_ключ_weatherstack&query=
```

### 4. Отримання API-ключів

- 🤖 **Telegram Bot Token** — створи бота через [@BotFather](https://t.me/BotFather) у Telegram.
- ✨ **Gemini API Key** — отримай на [Google AI Studio](https://aistudio.google.com/app/apikey).
- 🌍 **WeatherStack API Key** — зареєструйся на [weatherstack.com](https://weatherstack.com) і скопіюй свій ключ доступу.

### 5. Запуск бота

```bash
node bot.js
```

Бот запущено! Відкрий Telegram, знайди свого бота за іменем користувача та надішли будь-яке місто. 🌦️
