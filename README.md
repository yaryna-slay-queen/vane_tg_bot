# Vane 🌤️

**Vane** — це розумний Telegram-бот, який не просто сухо показує цифри прогнозу погоди, а перетворює їх на зрозумілі, турботливі та практичні поради для вашого дня. Завдяки інтеграції з моделлю штучного інтелекту Gemini 2.5 Flash, бот аналізує погоду в будь-якому куточку планети, підбирає актуальні рекомендації щодо одягу, попереджає про необхідність взяти парасольку та спілкується з вами як реальний помічник.

---

## 🎯 Цільова аудиторія (Target Audience)

Бот **Vane** створений для максимально широкого кола користувачів, оскільки погода впливає на щоденне життя кожного. Проте найбільшу користь він принесе:
* **Активним містянам та пішоходам:** людям, які багато пересуваються пішки або громадським транспортом і яким критично важливо знати, чи брати з собою парасольку, дощовик або сонцезахисні окуляри.
* **Мандрівникам та людям у відрядженнях:** тим, хто часто змінює локації та потребує швидкого аналізу погоди в новому місті без необхідності вивчати місцеві синоптичні сайти.
* **Батькам та піклувальникам:** яким щоранку потрібно швидко вирішити, як практично та комфортно одягнути дитину на прогулянку чи до школи.
* **Поціновувачам зручності та мінімалізму:** користувачам, які не хочуть завантажувати важкі мобільні додатки чи переглядати рекламу на сайтах погоди, а віддають перевагу отриманню персоналізованої інформації в один клік у звичному месенджері.

## 🛠️ Технологічний стек

Проєкт реалізований на базі сучасних інструментів розробки:
* **Мова програмування:** JavaScript (Node.js)
* **Платформа бота:** Telegram API (через фреймворк `telegraf`)
* **Штучний інтелект:** Google Gemini 2.5 Flash (`@google/genai`) — відповідає за пошук координат міст, інтерпретацію синоптичних даних та генерацію дружніх порад.
* **Погодні дані:** API сервісу `Weatherstack`
* **Мережеві запити:** `axios`
* **Конфігурація:** `dotenv`

## ⚙️ Покрокова інструкція із запуску

Дотримуйтесь цих кроків, щоб розгорнути та запустити бота локально.

### 1. Попередні вимоги
Переконайтеся, що на вашому комп'ютері встановлено **Node.js** (рекомендовано версію LTS).

### 2. Клонування проєкту та встановлення залежностей
Перейдіть у папку з проєктом та встановіть необхідні npm-пакети:

```bash
npm install @google/genai axios dotenv telegraf
```bash

### 3. Налаштування оточення (.env)
Створіть у кореневій папці проєкту файл з назвою .env за зразком файлу .env.example. Вам знадобиться заповнити такі змінні:
```bash
TOKEN=YOUR_TELEGRAM_TOKEN_HERE
Url=http://api.weatherstack.com/current?access_key=YOUT_WEATHERSTACK_API_KEY_HERE&query=
GEMINI_API_KEY=YOUR_GEMINI_API_KEY_HERE

Як отримати ключі:

TELEGRAM_BOT_TOKEN — створіть нового бота у Telegram через офіційного бота @BotFather.

GEMINI_API_KEY — отримайте безкоштовний або платний ключ у Google AI Studio.

WEATHERSTACK_API_KEY — зареєструйтеся на сайті Weatherstack та скопіюйте ваш API Access Key.

4. Запуск проєкту
Запустіть бота у консолі за допомогою команди:

Bash
node bot.js
Після успішного запуску перейдіть у Telegram, знайдіть свого бота та напишіть йому назву будь-якого міста (наприклад, Київ або London), щоб отримати розумний прогноз погоди!
